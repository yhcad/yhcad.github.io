# PicGo + GitHub 搭建个人图床工具

2019年在一次偶然的机会让我接触到Markdown文档，随后便疯狂地爱上了这种“轻量级标记语言”。笔记、文章、技术文档等等，我都想要使用Markdown去记录。我找到一款Markdown编辑器：VNote (opens new window)

在用Markdown写文档的时候经常需要截图，然后从剪贴板从直接黏贴放进文档，下面介绍PicGo小工具能满足这种操作，但是其实现原理不过是将截图存放在本机电脑上，所以，一旦文档中有了截图，将Markdown文档的迁移到别的电脑上展示的成本就大大增加（图片也要一并复制过去）。也是我在使用过程中唯一觉得不爽的一点。辛苦搜寻之下，终于让我找到了一种解决方案：PicGo+GitHub图床。

# PicGo图床工具的使用

-PicGo (opens new window)
图片上传+管理新体验，
PicGo在上传图片之后自动会将图片链接复制到你的剪贴板里，可选5种复制的链接格式。

PicGo目前支持了

  • 微博图床 v1.0
  • 七牛图床 v1.0
  • 腾讯云COS v4\v5版本 v1.1 & v1.5.0
  • 又拍云 v1.2.0
  • GitHub v1.5.0
  • SM.MS v1.5.1
  • 阿里云OSS v1.6.0
  • Imgur v1.6.0

快速上手 (opens new window)

注意

请确保你安装了 Node.js， 并且版本 >= 8。

提示

默认上传图床为SM.MS。PicGo上传之后，会自动将上传成功的URL复制到你的剪贴板，支持5种复制格式。

在众多的图床中，我选择的GitHub图床，其它类型的图床如果你们有兴趣的话可以试一下。
