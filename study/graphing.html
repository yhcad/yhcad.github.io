# Graphing Calculator 3D

# Graphing 介绍

三维图形计算器的想法于2007年来自于Saeid Nourian博士 (opens new window)，当时他还是加拿大渥太华大学的研究生。作为一名学生，他亲眼目睹了软件工具在理解科学和数学方面的有效性。他特别感到需要在3D中可视化数学方程和数据。由于他的专长是计算机图形学，他决定构建一个利用GPU的强大功能来快速计算和渲染3D图形的软件。他选择Java和Java3D作为开发平台，因为Java程序可以在流行的操作系统上无缝运行，包括Windows，Mac和Linux。结果是世界各地的50万学生和教师下载了这款Graphing Calculator 3D (opens new window) 功能强大的图形工具。


（Nourian博士拥有计算机科学博士学位，专攻虚拟现实和计算机图形学。他是Runiter公司生产的大多数软件产品的唯一作者。）
Graphing Calculator 3D是专为三维图像设计人员打造的一款专业计算工具，软件包含了在计算过程中用到的各种工具，包括在2D或3D中绘制任何隐式或显式方程。支持的坐标类型有：笛卡尔坐标，极坐标，圆柱坐标和球面坐标。可以绘制参数方程以及不等式。绘制三维散点图轮廓图生成价值表等，同时还支持自定义变量的变化值，运行后输入相应的参数就可以计算出需要的结果了，同时还拥有导入实时坐标进行计算的功能。

Graphing Calculator 3D 是大学生学习高等数学的好帮手。 更新版本 v7.1（2019年1月27日）官方下载 (opens new window)

# Graphing 调试

# 1.结缘Graphing

日前孩子在学习高等数学时，高数老师在课堂上使用 Graphing Calculator 3D软件来演示高等数学方程式，3D图形直观又容易理解，孩子让我帮助下载安装了这款工具。过了一周时间，孩子告诉我Graphing Calculator 3D 软件无法使用了。

注册过程也相当复杂，要在国外官方论坛 (opens new window)上注册写个文章，并发表关于Graphing Calculator 3D用法的问题建议，作者是否给你发送注册码还不一定的。
免费版试用期只能用3天时间，只可以作5个图形已经远远满足大学的《高等数学》学习了。专业版是无限制同时作图的，但每年的注册费用为199美元/年，如下图所示，但对于一个没有经济来源的大学生来说，软件注册费无疑是一笔昂贵的经济负担。

# 2.软件监视工具TU

利用Total Uninstall (opens new window) 专业版工具，对Graphing Calculator 3D程序运行时对系统都做了什么手脚进行监视。

免费版如何能继续使用呢？ 我在网上搜索了一遍，没有找到PJ的，决定自己动手DIY对Graphing Calculator 3D软件进行简单的调试。
通过Total Uninstall（简称Tu）对程序运行前后作两次系统快照的对比，发现注册表有个地方变化引起我的注意，如下图所示：

运行regedit打开注册表，定位到“HKEY_CURRENT_USER\Software\JavaSoft\Prefs\runiter\grapher”位置。

# 3.注册表位置

我发现Graphing Calculator 3D程序每运行一次后，在“HKEY_CURRENT_USER\Software\JavaSoft\Prefs\runiter\grapher”位置的注册表"uses"值会自动加1。 安装程序后每过一天，注册表days值就会自动加1，等到过了3天后，Graphing Calculator 3D无法打开再试用了。

# 4.批处理修改注册表

知道上面这个奥秘，我想PJ方法就是写一个bat批处理文件，让它在Graphing Calculator 3D程序启动前修改注册表uses值为0，days值为0，让这两个值永远等于0，这样程序就可以永久免费使用了，并且不用对Graphing Calculator 3D程序作任何修改就能达到免费效果。

赶快动手吧：在桌面新建一个文件：Graphing Calculator 3D.bat 复制如下内容后保存。

@ECHO OFF
REG ADD "HKEY_CURRENT_USER\Software\JavaSoft\Prefs\runiter\grapher" /V days /T REG_SZ /D 0 /F
REG ADD "HKEY_CURRENT_USER\Software\JavaSoft\Prefs\runiter\grapher" /V uses /T REG_SZ /D 0 /F
rem --修改注册表值，使uses=0，days=0，即已试用次数改为0，试用天数改为0

ping localhost -n 1
rem --间隔1秒钟时间再启动下面程序

start "Graphing Calculator 3D" "C:\Graphing Calculator 3D\Graphing Calculator 3D.exe"
rem --运行Graphing Calculator 3D.exe程序
EXIT

# 5.注意点

打开Graphing Calculator 3D程序只要双击桌面上的Graphing Calculator 3D.bat文件即可。 注意：我的Graphing Calculator 3D安装目录是C:\Graphing Calculator 3D，如果你是默认安装的，这个路径也要作相应调整的。
